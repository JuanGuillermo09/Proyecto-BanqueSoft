Cotizacion:

En este modulo el cliente puede cotizar el precio para un(1) evento escogiendo los servicios que desde el total se muestra en un aproximado ya que los precios estan sujetos en cambios por parte del administrador, adicionalmente tiene la opcion de generar la cotizacion realizada en un documento PDF para tener una base.

NOTAS:

Es importante rezaltar que lo precios que aqui se muestran son solo informativos (No se sostienen precios del modulo cotizacion al realizar un(1) contrato

Citas:

En este modulo el cliente podra agendar una(1) cita presencial con el administrador o persona encargada del banquete para una posible cotizacion de banquete para la ejecucion de un(1) evento.

NOTAS:

El cliente solo podra agendar una(1) cita y hasta que esta sea cumplida quedara inhabilitado.

Para pedir una(1) cita el cliente debe estar registrado en el sistema, en caso de no estarlo, se debera registrar (En el modulo encontrado la opcion).
